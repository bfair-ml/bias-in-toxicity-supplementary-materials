# Supplementary Materials – Balancing the Scales: Addressing Gender Bias in Social Media Toxicity Detection

This repository contains the **supplementary materials** for the paper:
**"Balancing the Scales: Addressing Gender Bias in Social Media Toxicity Detection"**.

## Contents

[`Appendix`](./materials/Appendix.pdf):
A single PDF file that contains the algorithms used in the study, with detailed pseudocode and descriptions referenced in the paper.
